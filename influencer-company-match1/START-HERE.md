## 🚨 Troubleshooting

### Login Not Working?
1. Check backend logs
2. Verify DATABASE_URL is set
3. Try manual deploy

### Service Spinning Down?
1. This is normal on free tier
2. First request wakes it up
3. Takes ~30 seconds

### Tables Not Created?
1. Check logs for errors
2. Try manual deploy
3. Verify DB_SYNCHRONIZE=true

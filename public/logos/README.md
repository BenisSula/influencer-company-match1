# Logo Assets

This directory contains logo images for the "As seen on" carousel on the landing page.

## Required Logos

Add the following logo files (SVG format recommended):

1. `techcrunch.svg` - TechCrunch logo
2. `forbes.svg` - Forbes logo
3. `wired.svg` - Wired logo
4. `theverge.svg` - The Verge logo
5. `mashable.svg` - Mashable logo

## Specifications

- Format: SVG (preferred) or PNG
- Size: Height should be ~40px
- Color: Grayscale or brand colors (will be converted to grayscale with CSS)
- Background: Transparent

Until real logos are added, you can use placeholder images or text-based logos.
